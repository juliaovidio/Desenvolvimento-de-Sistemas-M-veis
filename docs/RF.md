## Requisitos Funcionais do Sistema

### RF01 – Permissão de localidade:
O sistema deve pergunatr se o usuario permiti que ele utilize sua localização.

### RF02 – Compartilhamento de Conteúdo:
O sistema deve permitir que o usuário compartilhe informações da sua localidazação para o gerente quando apertar o botão de atualizar. 

### RF03 – Histórico de Atividades:
O sistema deve registrar e permitir que o usuário visualize um histórico detalhado de todas as informações do seu trabalho(trajeto) que irá realizar.
